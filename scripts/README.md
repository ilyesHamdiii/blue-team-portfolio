# Automation Scripts

This folder contains small scripts developed to assist with log analysis, alert enrichment, or repetitive investigative tasks.

---

## Objectives

- Improve efficiency during investigations
- Reduce manual parsing of logs
- Demonstrate practical scripting ability

---

## Guidelines

Each script includes:

- A short description of its purpose
- Usage instructions
- Any assumptions or limitations

Scripts are written for the lab environment and are not intended for production use.
