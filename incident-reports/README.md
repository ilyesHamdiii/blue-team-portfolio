# Incident Response Reports

This folder contains documented investigations of simulated security events observed in the lab environment.

Each report reflects a structured approach to alert triage and log-based investigation.

---

## Report Structure

Each incident report includes:

- Executive summary
- Alert or detection source
- Timeline of relevant activity
- Analysis and findings
- Impact assessment
- Response and containment steps
- Lessons learned
- Identified limitations or data gaps

---

## Purpose

The goal is to demonstrate:

- Analytical reasoning under incomplete information
- Log correlation and timeline reconstruction
- Clear and objective documentation
- Practical response recommendations

The reports are written in a format suitable for internal SOC documentation.
